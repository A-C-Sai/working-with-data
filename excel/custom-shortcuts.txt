Custom Shortcuts MacOS
------------------------

1. command+shift+t ---> Auto Sum
2. control+u ---> Toggle edit/enter mode
3. control+ + + ` ----> show formulas
4. command+t -----> toggle cell referencing
5. command+shift+p ----> percent formatting
6. command+shift+n -----> format cells window


7. fn+control+up/down arrow ---> move between worksheets
8. fn+control+left/right arrows ----> move between first visible cell or last used cell

9. control+option+return ---> line break

10. command/control+shift++ ------> Add Column/Row
11. command/control+- ------> Delete Column/Row

12. control+space ---> select entire col
13. shift+space ----> select entire row
14. To rearrange cols: select whole column then hold shift while moving to desired place
15. control/command+shift+down ----> Select the range from the active cell to the last cell before the next blank cell in a column.

16. command+up/down/right/left ---> Select the last cell before the next blank cell in a column.



17. Group selected cell(s) into rows or columns -----> Cmd+Shift+K
18. Ungroup selected cell(s) into rows or columns -------> Cmd+Shift+J 
19. Hide rows ---> Ctrl+9
20. Unhide rows ------> Ctrl+Shift+9
21. Hide columns -------> Ctrl+0
22. Unhide columns -------> Ctrl+Shift+0


23. Copy/duplicate a sheet -----> option+drag 
24. command+o
25. command+n

26. control+command+v ---> open paste special window
27. return ---> paste
28. control+command+v+v+return ------> paste values
29. Keeps the same cell selected after entering a formula/value -----> Ctrl+Return

30. control+G -----> Open the Go To window
31. Turn Filters On/Off ------> Cmd+Shift+F

33. select visible cells (when range is selected) ---> control+command+;
34. control+H ---> find and replace window
