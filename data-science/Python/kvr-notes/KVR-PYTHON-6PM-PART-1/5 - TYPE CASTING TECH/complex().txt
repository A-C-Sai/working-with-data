		========================================
				4. complex()
		========================================
=>complex() is used for Converting Any Type of Possible Value into complex type value.
=>Syntax:varname=complex(int / float / bool / str int/ str float/ str bool/ str complex/ str)
------------------------------------------------------------------------------------------------------------------------
Example: int----->complex---->Possible
------------------------------------------------------------------------------------------------------------------------
>>> a=10
>>> print(a,type(a))--------10 <class 'int'>
>>> b=complex(a)
>>> print(b,type(b))--------(10+0j) <class 'complex'>
------------------------------------------------------------------------------------------------------------------------
Example: float----->complex---->Possible
------------------------------------------------------------------------------------------------------------------------
>>> a=2.5
>>> print(a,type(a))--------------2.5 <class 'float'>
>>> b=complex(a)
>>> print(b,type(b))-------------(2.5+0j) <class 'complex'>
------------------------------------------------------------------------------------------------------------------------
Example: bool----->complex---->Possible
------------------------------------------------------------------------------------------------------------------------
>>> a=True
>>> print(a,type(a))--------True <class 'bool'>
>>> b=complex(a)
>>> print(b,type(b))-------(1+0j) <class 'complex'>
-----------------------------------------------------------------------------------------------------------------------
Examples:   str ------>complex
-----------------------------------------------------------------------------------------------------------------------
Case-1:  str int--------->Complex--->Possible
-----------------------------------------------------------------------------------------------------------------------
>>> a="10"
>>> print(a,type(a))-------------10 <class 'str'>
>>> b=complex(a)
>>> print(b,type(b))----------(10+0j) <class 'complex'>
-----------------------------------------------------------------------------------------------------------------------
Case-2:  str float--------->Complex--->Possible
-----------------------------------------------------------------------------------------------------------------------
>>> a="2.3"
>>> print(a,type(a))------------2.3 <class 'str'>
>>> b=complex(a)
>>> print(b,type(b))---------(2.3+0j) <class 'complex'>
-----------------------------------------------------------------------------------------------------------------------
Case-3:  str bool--------->Complex--->Not Possible
-----------------------------------------------------------------------------------------------------------------------
>>> a="True"
>>> print(a,type(a))-------------True <class 'str'>
>>> b=complex(a)-------------ValueError: complex() arg is a malformed string
-----------------------------------------------------------------------------------------------------------------------
Case-4:  str complex--------->Complex---> Possible
-----------------------------------------------------------------------------------------------------------------------
>>> a="2.3+5.6j"
>>> print(a,type(a))----------2.3+5.6j <class 'str'>
>>> b=complex(a)
>>> print(b,type(b))---------(2.3+5.6j) <class 'complex'>
-----------------------------------------------------------------------------------------------------------------------
Case-5:  Pure str --------->Complex---> Not Possible
-----------------------------------------------------------------------------------------------------------------------
>>> a="Python"
>>> print(a,type(a))-----------------Python <class 'str'>
>>> b=complex(a)-------------ValueError: complex() arg is a malformed string
-----------------------------------------------------------------------------------------------------------------------
