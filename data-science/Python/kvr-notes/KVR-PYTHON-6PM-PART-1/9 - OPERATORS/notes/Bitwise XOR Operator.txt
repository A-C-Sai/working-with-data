		===========================================
				Bitwise XOR Operator ( ^ )
		===========================================
=>Syntax:     var name= Var1 ^ Var2
=>The Functionality of Bitwise XOR Operator (^ ) is expressed by using the following Truth table.
=>Similar Bits are Zero and Dissimilar Bits are 1
		----------------------------------------------------------------------
			Var1		Var2		Var1 ^ Var2
		----------------------------------------------------------------------
			0		1			1

			1		0			1

			0		0			0

			1		1			0
		----------------------------------------------------------------------

Examples:
---------------------
>>> a=5
>>> b=6
>>> c=a^b
>>> print(c)------------3
>>> a=3
>>> b=4
>>> c=a^b
>>> print(c)---------7
>>> a=10
>>> b=15
>>> c=a^b
>>> print(c)-----------5
